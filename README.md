# Drone Rescue AI Simulator

Real-time AI-powered drone coordination system for search & rescue operations.

## Live Demo
👉 [Open Simulator](https://ТВО-НИК.github.io/drone-rescue-ai/drone-rescue.html)

## Features
- Claude AI (Haiku) selects optimal rescue drone in real time
- Live weather data from coordinates via wttr.in
- 5 autonomous drones with physics simulation
- Full telemetry: battery, wind limits, IP rating, payload, range
- Runs entirely in a single HTML file

## Tech Stack
Vanilla JS · Canvas API · Claude Haiku API · wttr.in API

## How to Use
1. Open the simulator
2. Click **RUN CLAUDE AI ANALYSIS**
3. Click **DISPATCH** to send the optimal drone
